# 🌳 OliBook

> Web App dedicada al mundo del olivar. Podrás tener un cuaderno de campo sin necesidad de papel y hoja, todo a la altura de tu dispositivo móvil.

---

## 🚀 Características Principales

* **Un Inicio con lo Principal:** Una cálida bienvenida, un rápido vistazo a los últimos trabajos y acceso a las funciones principales de la app.
* **Gestión de Parcelas:** Control visual y detallada de cada una de tus parcelas. Podrás crearlas de manera manual, o mediante importación con SIGPAC.
* **Registro de Actividades:** Registra y visualiza un calendario con los días trabajados, como las actividades realizadas en tus parcelas.
* **Perfil:** Donde podrás realizar los cambios en tu perfil deseados, así como trabajar la configuración de la app para ajustar a tus necesidades.

## 🛠️ Tecnologías Utilizadas

*   [Ionic Framework](https://ionicframework.com/) - Frontend híbrido móvil.
*   [Angular](https://angular.dev/) - Framework de desarrollo web.
*   [TypeScript](https://www.typescriptlang.org/) - Lenguaje de programación.
*   [SASS / SCSS](https://sass-lang.com/) - Estilos avanzados y modulares.
*   [SIGPAC](https://sigpac-hubcloud.es/) - API de SIGPAC que permite consultar geometrías y propiedades en suelo español.
